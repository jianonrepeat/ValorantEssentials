# ValorantEssentials

A powerful utility for VALORANT players that provides essential tools including stretched resolution support and custom content management.

## Features

- **Stretched Resolution**: Easily set and manage custom resolutions for VALORANT
- **One-Click Setup**: Simple interface for quick configuration
- **Automatic Resolution Management**: Automatically switches to your preferred resolution when VALORANT launches
- **Custom Content Support**: Install and manage custom game assets

## Requirements

- Windows 10/11
- VALORANT installed
- Administrator privileges
- PowerShell 5.1 or later

## Installation

1. Download the latest release or clone this repository
2. Run `run.bat` as Administrator
3. Follow the on-screen instructions

## Usage

1. **Stretched Resolution**
   - Enter your desired resolution (e.g., 1440x1080)
   - Click "Start Stretched Res Launcher"
   - Launch VALORANT

2. **Custom Content**
   - Click "Install/Update 'Blood Paks'" to install custom game assets
   - Follow any on-screen prompts

## Troubleshooting

- If you encounter permission errors, ensure you're running as Administrator
- If resolution changes don't apply, make sure VALORANT is running in Fullscreen or Borderless Windowed mode
- For custom content issues, verify your VALORANT installation path is correct

## Safety

This tool only modifies VALORANT's configuration files and does not interact with game memory or Vanguard anti-cheat. However, use custom content at your own risk.
